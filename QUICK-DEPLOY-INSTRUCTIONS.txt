METHOD 1: Using cPanel File Manager (EASIEST)
========================================

   - Visit https://www.hrleaveassist.com/

   - Press Ctrl+F5 to clear cache

========================================
METHOD 2: Using cPanel Git Version Control
========================================

Step 1: Push changes to GitHub
   - Run update-repo.bat on local machine

Step 2: In cPanel Git Version Control
   - Click "Manage" on your repository
   - Click "Pull or Deploy" tab
   - Click "Update from Remote"
   - Click "Deploy HEAD Commit"

Step 3: The .cpanel.yml file will automatically:
   - Copy all files from repository to public_html
   - This happens automatically when you click Deploy

Step 4: Verify
   - Visit https://www.hrleaveassist.com/
   - Press Ctrl+F5 to clear cache

========================================
TROUBLESHOOTING
========================================

Issue: Still seeing directory listing
Solution: 
   - Make sure index.php exists in public_html
   - Check file permissions (should be 644)
   - Check .htaccess file exists

Issue: 404 errors
Solution:
   - Verify all files are in public_html (not in a subfolder)
   - Check that paths in config/app.php are correct

Issue: White page / PHP errors
Solution:
   - Check error_log file in public_html
   - Verify PHP version is 7.4 or higher
   - Check database connection in config/local.php
